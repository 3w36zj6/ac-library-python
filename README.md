# ac-library-python

[日本語のREADME](README.ja.md)

## Description

ac-library-python is a Python port of [AtCoder Library (ACL)](https://atcoder.jp/posts/517). For more information, see [References](#References).

**Note: There is no test code at this time.**

## How to contribute

Paste the following commands at a Terminal prompt.

```md
$ mkdir your_dir_name
$ cd your_dir_name
$ git clone git@github.com:not522/ac-library-python.git
```

1. Create an issue or a draft PR before you start working.
2. Make a PR and pass tests.
